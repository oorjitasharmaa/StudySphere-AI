# 🚀 StudySphere AI

StudySphere AI is an AI-powered learning platform that helps students summarize study notes, generate quizzes, and interact with their notes using AI.

## ✨ Features

* 📚 AI Notes Summarization
* 📝 AI Quiz Generation
* 💬 Chat with Notes
* 🔐 User Authentication (Signup/Login)
* 📄 PDF Upload Support
* 📊 Dynamic Dashboard
* ☁️ Cloud Deployment on Render

## 🛠️ Tech Stack

* Python
* Flask
* SQLite
* HTML
* CSS
* Groq API
* PyPDF2

## ⚙️ Installation

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Add your Groq API key

GROQ_API_KEY=your_api_key

4. Run the application

python app.py
